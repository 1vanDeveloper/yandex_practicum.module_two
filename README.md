# Работа по Yandex Practicum Middle Java-разработчик

## Запуск приложения в Docker
Выполните команду в корне проекта:
```shell
docker-compose up --force-recreate --renew-anon-volumes -d
```
Поднимется контейнер СУБД PostgreSQL необходимый для интеграционного тестирования и рабюоты приложения в Docker.
Docker-compose настроен так, что при запуске контейнера PostgreSQL будет инициализирована схема данных и некоторые данные для тестирования.
Также соберется и настроется приложение, сконфигурированное под указанную выше БД.

## Сборка и запуск приложения
Выполните команду в корне проекта:
```shell
./gradlew bootJar
java -jar build/libs/market-1.0-SNAPSHOT.jar
```
